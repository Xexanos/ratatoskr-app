# Upstream logo source assets

Base artwork the Ratatoskr logo is derived from. All were published under the
**CC0 1.0** license (no attribution required; recorded here anyway for provenance):

- `squirrel-cute-squirrel-svgrepo-com.svg` — shape baseline for the mark.
  Source: https://www.svgrepo.com/svg/307552/squirrel-cute-squirrel (CC0)
- `squirrel-svgrepo-com.svg` — multi-tone color-separation reference.
  Source: https://www.svgrepo.com/svg/275570/squirrel (CC0)
- `celtic-knot-frame-svgsilh-2766191.svg` — knot frame reference/base.
  Source: https://svgsilh.com/de/image/2766191.html (CC0, vectorized Pixabay asset)

The derived logo (mark, icon, lockup) is part of this project and licensed
GPL-3.0-or-later like the rest of the repository.
